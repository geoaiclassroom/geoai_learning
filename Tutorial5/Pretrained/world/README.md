## To run the notebooks in this tutorial, I have manually downloaded the world shapefile and placed it here on my local machine.

To do the same, you can download the shapefile from the NaturalEarthData page:

https://www.naturalearthdata.com/http//www.naturalearthdata.com/download/110m/cultural/ne_110m_admin_0_countries.zip

You need to unzip the files inside the world folder so that the "ne_110m_admin_0_countries.shp" file is directly found in "world" folder.
