## Pour exécuter les notebooks de ce tutoriel, j'ai téléchargé manuellement le shapefile mondial et l'ai placé ici sur ma machine locale.

Pour faire de même, vous pouvez télécharger le shapefile depuis la page NaturalEarthData :

https://www.naturalearthdata.com/http//www.naturalearthdata.com/download/110m/cultural/ne_110m_admin_0_countries.zip

Vous devez décompresser les fichiers dans le dossier world afin que le fichier "ne_110m_admin_0_countries.shp" se trouve directement dans le dossier "world".
